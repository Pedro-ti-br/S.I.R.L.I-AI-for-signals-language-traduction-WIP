📝 Descrição

O Sirli é um projeto focado em acessibilidade que utiliza visão computacional e modelos de IA para traduzir gestos da Libras em texto ou voz. O objetivo é quebrar barreiras de comunicação entre surdos e ouvintes de forma fluida e tecnológica.
✨ Funcionalidades Planejadas

    Reconhecimento em Tempo Real: Captura de movimentos via webcam.

    Tradução Contextual: Processamento de frases completas, não apenas sinais isolados.

    Interface Intuitiva: Foco total na experiência do usuário (UX) para garantir agilidade na comunicação.

🛠️ Tecnologias (Stack)

    Linguagem: Python

    IA/Visão Computacional: OpenCV, MediaPipe ou TensorFlow (em definição).

    Interface: Flask / FastAPI (Web) ou interface Desktop.

    Ambiente: Desenvolvido em Linux (CachyOS).

🇺🇸 English
📝 Description

Sirli is an accessibility-focused project that leverages computer vision and AI models to translate Sign Language (Libras) gestures into text or speech. The goal is to break communication barriers between deaf and hearing individuals through seamless technology.
✨ Planned Features

    Real-time Recognition: Gesture capture via webcam.

    Contextual Translation: Processing of complete sentences rather than isolated signs.

    Intuitive Interface: Full focus on User Experience (UX) to ensure fast and effective communication.

🛠️ Tech Stack

    Language: Python

    AI/Computer Vision: OpenCV, MediaPipe, or TensorFlow (under evaluation).

    Interface: Flask / FastAPI (Web) or Desktop UI.

    Environment: Developed on Linux (CachyOS).

🚀 Roadmap / Próximos Passos
